My Code-OSS APT Repo

What I did is just build the official Code - OSS repo on GitHub and update my APT repo.

How To Install

First, install my key.
curl -sSL https://tuftstar.netlify.app/sw.key | sudo apt-key add -
Then, add the repo into your sources.list.
sudo bash -c 'echo "deb https://tuftstar.netlify.app/codeossrepo /" >> /etc/apt/sources.list'
The last step, install Code-OSS.
sudo apt update
sudo apt install code-oss

note: I can't update this repo every day, but I'll do my best to keep it up-to-date. NOTE: if it shows this error message: /usr/share/code-oss/bin/../code-oss: error while loading shared libraries: libffmpeg.so: cannot open shared object file: No such file or directory, then find libffmpeg.so in your computer(or on the Internet, it can be anywhere), and copy it into /usr/share/code-oss.

How to use the Visual Studio Marketplace in Code-OSS

By default, Code - OSS does not support Visual Studio Marketplace. To fix that, add these into ~/.config/Code - OSS/product.json (or /usr/share/code-oss/resources/app/product.json, but it will be rewritten after updating/reinstalling):
"extensionsGallery": {
    "serviceUrl": "https://marketplace.visualstudio.com/_apis/public/gallery",
    "cacheUrl": "https://vscode.blob.core.windows.net/gallery/index",
    "itemUrl": "https://marketplace.visualstudio.com/items",
    "resourceUrlTemplate": "https://{publisher}.vscode-unpkg.net/{publisher}/{name}/{version}/{path}"
}
You can also use my package code-oss-marketplace. It works on my computer well.
sudo apt install code-oss-marketplace
