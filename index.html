My Code-OSS APT Repo

What I did is just build the official Code - OSS repo on GitHub and update my APT repo.

How To Install

First, install my key.
curl -sSL https://tuftstar.netlify.app/sw.key | sudo apt-key add -
Then, add the repo into your sources.list.
sudo bash -c 'echo "deb https://tuftstar.netlify.app/codeossrepo /" >> /etc/apt/sources.list'
Finally, install Code-OSS.
sudo apt update
sudo apt install code-oss

note: I can't update this repo every day, but I'll my best to keep it up-to-date.

How to use the Visual Studio Marketplace in Code-OSS

By default, Code - OSS does not support Visual Studio Marketplace. To fix that, add these into /usr/share/code-oss/resources/app/product.json:
"extensionsGallery": {
    "serviceUrl": "https://marketplace.visualstudio.com/_apis/public/gallery",
    "cacheUrl": "https://vscode.blob.core.windows.net/gallery/index",
    "itemUrl": "https://marketplace.visualstudio.com/items",
    "resourceUrlTemplate": "https://{publisher}.vscode-unpkg.net/{publisher}/{name}/{version}/{path}",
    "recommendationsUrl": "https://az764295.vo.msecnd.net/extensions/workspaceRecommendations.json.gz"
}
